# 🎮 Games Hub

**Games Hub** is a simple, browser-based game hosting site that lets users access and play a collection of web games from one place.

It’s designed to be lightweight, easy to deploy, and fully compatible with **GitHub Pages**, making it a convenient way to organize and share browser games.

---

## 📖 About

This project acts as a **central hub for HTML5/browser games**, where all games are loaded dynamically from a JSON file. Instead of hardcoding games into the page, the site builds its game library automatically — making it easy to expand and maintain.

Because it runs entirely in the browser and can be hosted on GitHub Pages, it can work in environments with limited software access (like school-issued devices), depending on network restrictions.

---

## ⚙️ How It Works

- A `games.json` file stores the list of available games  
- The frontend reads this file and generates the UI dynamically  
- Each game is loaded via:
  - Local files, or
  - Embedded links (iframes)

This makes adding new games as simple as editing one file.

---

## 🎯 Purpose

- Provide a **centralized place** to organize browser games  
- Allow quick access without downloads or installs  
- Demonstrate a **simple JSON-driven web app structure**  
- Serve as a beginner-friendly project for learning web development
